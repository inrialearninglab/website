_Classes start : 11 mai 2015_

This course describes the computational principles of binaural hearing; <!--more-->how these principles could be implemented on a robot head and how they could lead towards robust interaction capabilities.

This MOOC was created by Inria’s MoocLab, as part of the uTOP project ( [http://utop.fr/](http://utop.fr/) - [http://utop.inria.fr/](http://utop.inria.fr/)). uTOP is an IDEFI project that aims to create a demonstrator for increasing research visibility through out training.
